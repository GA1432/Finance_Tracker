# Finance Tracker - Product Overview

## Purpose
A full-stack personal finance tracking application that allows users to log, view, and manage their expenses. The app provides a dashboard for monitoring spending and supports user authentication.

## Key Features
- **Expense Management**: Create, list, and delete expense records with amount, category, description, and date
- **Spending Analytics**: Aggregate total spending via backend service (dashboard-ready)
- **User Authentication**: Sign-in and sign-up flows via dedicated auth routes
- **Categorized Expenses**: Tag expenses by category (e.g., "Online Food") with optional descriptions
- **Chronological Ordering**: Expenses returned newest-first for intuitive display

## Target Users
Individual users who want a simple, self-hosted tool to track personal or household expenses.

## Use Cases
- Log daily expenses with category and optional notes
- View all past expenses sorted by date
- Delete incorrect or duplicate entries
- Monitor total spending for budgeting purposes
- Register and authenticate to access personal expense data
